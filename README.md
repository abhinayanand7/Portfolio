# 🌐 Personal Portfolio – Abhinay Anand

🚀 A modern, responsive personal portfolio website showcasing my skills, projects, and experience as a Full Stack Developer.

## 🔗 Live Demo

👉 https://portfolio-abhinay.netlify.app/

## 📂 GitHub Repository

👉 https://github.com/abhinayanand7/Portfolio

## ✨ Features

* 🎯 Clean and modern UI design (Glassmorphism style)
* 📱 Fully responsive (Mobile + Tablet + Desktop)
* 🌙 Light/Dark mode toggle
* 🧑‍💻 About section with education & profile
* 🛠 Skills & Tech Stack visualization
* 🚀 Projects showcase with live preview
* 💼 Experience timeline section
* 📬 Contact form UI
* ⚡ Smooth scrolling & animations

---

## 🛠 Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **Styling:** Custom CSS (Flexbox, Grid)
* **Deployment:** Netlify
* **Version Control:** Git & GitHub
